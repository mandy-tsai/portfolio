# Portfolio 改版規劃文檔

## 一、設計方向總覽

### 核心風格：Editorial Magazine（雜誌編排風）

從參考圖片中提煉出的設計關鍵字：
- **黑白為主調**，偶爾使用單一強調色（如橙色或黃色）用於小型標籤或互動元素
- **大量留白**，以淺灰/米白（#F2F0ED）作為背景，而非深黑
- **超大字體排版**，錯落不對稱，文字本身就是視覺元素
- **編號系統**（01, 02, 03...）貫穿全站，增加秩序感
- **混搭字體**：襯線體（Serif）用於大標題營造雜誌感 + 無襯線體（Sans-serif）用於內文
- **網格系統**：嚴謹的底層 grid，但內容刻意突破格線製造張力

## 二、色彩規劃

| 角色         | 色碼      | 用途                         |
|------------|---------|----------------------------|
| 背景色        | #F2F0ED | 全站主要背景（暖米白）                |
| 主文字色       | #1A1A1A | 標題、主要內容文字                  |
| 次要文字色      | #6B6B6B | 描述文字、標籤、小字                 |
| 邊線/分隔線     | #D4D0CB | 區塊分隔線、卡片邊框                 |
| 強調色（極少量使用） | #FF4D00 | hover 狀態、目前頁面標記、小圓點裝飾      |
| 深色區塊       | #1A1A1A | 作品卡片內的文字說明區塊（黑底白字，製造反差）    |

---

## 三、字體規劃

| 用途     | 字體                           | 風格       |
|--------|------------------------------|----------|
| 大標題    | **Playfair Display** 或 **DM Serif Display** | 襯線體，雜誌感  |
| 次標題/導航 | **Space Grotesk**            | 幾何無襯線體   |
| 內文     | **Inter**                    | 易讀無襯線體   |
| 編號/裝飾  | **Space Grotesk** 粗體         | 大號編號用    |

### 字級系統

- Hero 標題：`clamp(4rem, 10vw, 10rem)`（極大，佔滿視覺）
- 區塊標題：`clamp(2.5rem, 5vw, 5rem)`
- 卡片標題：`1.5rem ~ 2rem`
- 內文：`1rem ~ 1.125rem`
- 小標籤/編號：`0.75rem ~ 0.875rem`，tracking 加寬

## 五、共用組件規劃

### 5.1 Navbar
- 固定頂部，但**不使用** mix-blend-difference
- 淺色背景上用深色文字
- 佈局：左側「Logo」，右側「Works / About / Contact」
- 左下角可放狀態文字「Open for offers & collaborations —」
- 右下角可放「Portfolio — Vol.1」

### 5.2 滾動跑馬燈 (Marquee)
- 大號粗體字，描邊或半透明效果
- 無限水平滾動動畫
- 用於區塊之間的視覺分隔

### 5.3 作品卡片
- **Grid 版（首頁）**：3 欄 Grid，每張卡片含大圖 + 底部資訊（名稱 / 類型 / 年份 / 編號）
- **時間軸版（作品集頁）**：左右交替，帶透視旋轉的截圖 + 黑色文字區塊

### 5.4 Footer
- 三欄佈局
- 左：版權 + Email + Back to Top
- 中：裝飾性 Logo 圖形
- 右：社群連結列表

## 六、互動與動畫規劃

| 元素            | 動畫效果                                  |
|---------------|---------------------------------------|
| 頁面進場          | 內容由下往上淡入（stagger 交錯延遲）               |
| 滾動跑馬燈         | CSS 無限水平位移動畫                          |
| 作品卡片 hover    | 圖片微幅放大 (scale 1.05) + 覆蓋色層淡入          |
| 拱門卡片 hover    | 圖片上移 + 底部出現專案名稱                       |
| 個人照片 hover    | 像素化效果消失，顯示清晰照片（canvas 或 CSS filter）   |
| 編號            | 滾動時數字淡入 + 微幅上移                        |
| 連結 hover      | 底線從左到右展開                               |
| 按鈕 hover      | 背景色反轉（白底黑字 ↔ 黑底白字）                    |
| 頁面切換          | 內容淡出 → 新頁面內容淡入                        |
| 旋轉圓形文字        | 持續緩慢旋轉的圓形排列文字（如「Hover over the photo」）|

## 七、響應式斷點

| 斷點           | 寬度         | 調整                                  |
|--------------|------------|-------------------------------------|
| Mobile       | < 768px    | 單欄佈局、作品 Grid 改為 1 欄、標題字級縮小         |
| Tablet       | 768-1024px | 作品 Grid 改為 2 欄、兩欄佈局維持但比例調整          |
| Desktop      | > 1024px   | 完整 3 欄 Grid、所有裝飾元素顯示                |

## 八、實作優先順序

1. **Phase 1：基礎改版** — 色彩反轉（深色→淺色背景）、字體替換、Navbar/Footer 改版
2. **Phase 2：首頁重構** — Hero 大字排版、作品 Grid（3欄+編號）、Working Approach 區塊
3. **Phase 3：作品集頁** — 時間軸式卡片佈局、透視旋轉截圖、黑色文字區塊
4. **Phase 4：關於我頁** — 大 Serif 標題、拱門卡片列、像素化照片效果
5. **Phase 5：細節打磨** — 所有 hover 動畫、旋轉文字、滾動動畫、響應式調整
